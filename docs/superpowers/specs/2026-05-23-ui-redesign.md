# UI 重设计方案 — 微缩模型涂装辅助工具

> 迭代目标：用 Open Design Discord 设计系统重构前端界面，解决单页面拥堵、3D 预览局促、光源操作不灵活的问题。

---

## 1. 问题陈述

当前实现的所有功能挤在单页 Dashboard 中，漆料库变成侧边栏、3D 预览只有 400px 固定宽度、光源只能通过 XYZ 滑条调节。不符合用户实际操作流程，也不符合 Open Design 的规范用法。

## 2. 页面结构

| 路由 | 页面 | 说明 |
|------|------|------|
| `/paints` | 漆料库 | 独立全屏页面，完整 CRUD + 拍照取色 + 品牌筛选 |
| `/workspace` | 工作区 | 混色引擎 + 3D 预览左右分栏，核心操作页面 |
| `/recipes` | 配方库 | 已保存配方浏览/删除 |

默认登录后跳转 `/workspace`。

## 3. 导航设计 — 折叠侧边栏

- **默认收起**：左侧 56px 宽图标列（🎨漆料 🖌️工作 📋配方）
- **展开**：点击 ≡ 或图标，侧边栏从左侧滑出 220px，覆盖部分主内容
- **关闭**：点击外部区域或 ✕
- **设计系统**：Discord 暗色主题（bg #313338, sidebar #2b2d31, icon hover #3f4147）

## 4. 工作区布局（/workspace）

```
┌─────────────────────┬──────────────────────────┐
│ 混色面板 (~320px)    │  3D 预览 (flex: 1)        │
│                     │                          │
│  目标色选取          │  材质球 / STL 模型       │
│  (图片/色轮/输入)    │  地面反射平面            │
│  [计算混色]          │                          │
│                     │  光源栏 [+添加光源]       │
│  候选列表 TOP-10     │  ● 光源1 [x] 强度 ▬     │
│                                                     │
│  色轮 (可折叠)      │  材质: color roughness   │
│  互补·三角·类比     │        metalness         │
└─────────────────────┴──────────────────────────┘
```

## 5. 3D 预览增强

| 功能 | 描述 |
|------|------|
| **地面反射平面** | 灰色半透明 plane 置于 y=0，接收光源形成漫反射二级高光，模拟桌面/展台 |
| **多光源** | 默认 1 个光源；[+添加] 新增；每个光源一个可拖拽小球标记位置；选中高亮；[x] 删除 |
| **光源预设** | 顶光/侧光/正光快捷按钮，新光源默认正上方 |
| **光源强度** | 每个光源独立强度滑块 (0-3) |
| **预览比例** | 3D 画布占工作区右侧全部剩余空间（≥60% 宽度） |

## 6. 漆料库页面（/paints）

全屏独立页面：表格列表 + 品牌筛选 + +添加按钮 + PaintForm 弹窗 + PhotoColorPicker。

## 7. 配方库页面（/recipes）

全屏独立页面：RecipeCard 网格 + 删除确认。

## 8. Open Design 集成

- **设计系统**：Discord
- **Skill**：`frontend-design`（通过 Open Design MCP 生成 UI 原型）
- **实际实现**：React + TypeScript + Discord 暗色 CSS

## 9. 验证标准

- `npm run build` 零 TypeScript 错误
- 侧边栏折叠/展开动画流畅
- 3D 预览地面反射+多光源可操作
- 后端 API 不变，所有现有接口持续可用
