# 项目简报 — 微缩模型涂装辅助工具

> 我是 Airport233，这份文档是我向新 agent session 介绍项目的第一手材料。用它替代口头描述，让新 session 从源头走一遍 Superpowers brainstorming 流程。

---

## 我是谁

我是微缩模型涂装玩家（战锤、高达、D&D 微缩模型），非美术专业背景。我有一定涂装经验，正在追求进阶技法，特别是 NMM（非金属金属质感）。

## 我想做什么

一个 **Web 工具**，帮助微缩模型涂装玩家：
1. 录入手头已有漆料的真实颜色（拍照取色，消除卖家秀色差）
2. 从任意图片取目标色，计算用手头漆混合出目标色的方案
3. 在 3D 材质球（以及上传的 STL 模型）上预览颜色在不同光泽度、金属度、光源角度下的效果

## 核心痛点

1. **照抄教程偏色**：视频/照片中的漆色受灯光和底色影响，买了同款漆涂完发现颜色不对
2. **自混代换翻车**：不想为某个颜色买新漆，用手头已有漆混色替代，凭感觉混不准、容易"发灰"

## 我的初步设想

### 核心模块

| 模块 | 做什么 |
|------|--------|
| 漆料库 | 录入我手头的漆：品牌（GW/AV/AK/GSW/Scale75 等主流）、色号、RGB（自己拍照取色）。基本的增删改查 |
| 混色引擎 | 图片取目标色 → 用库里已有漆计算混合方案 → 多个候选按偏差排序 → 三原色方案作为参考基线 |
| 3D 预览 | 默认几何体（球/立方体/圆柱）+ 可上传 STL → 设置颜色、粗糙度、金属度 → 调节光源 → 预览涂装效果。NMM 技法非常依赖光影参考 |
| 配方管理 | 保存混色结果，方便以后回看 |

### 混色系统的偏好

- 按**整数份数**（1:1、1:2、2:1:1 这种），不是百分比。实际操作中按"滴"或"刷量"来混，最多 3 种漆、总份数不超过 5
- 当某漆占比极小时自动显示"少量"，不是预设哪些漆是极端色
- MVP 阶段所有漆当不透明处理，后期可以再考虑透明度

### 3D 预览的偏好

- 不是照片级渲染，是材质球参考工具
- 多光源支持，光源颜色可选（NMM 中不同色温的影响很重要）
- 模型应该能独立旋转和调高度，因为 STL 文件建模坐标系可能不一致
- 需要一个地面参考（网格），让模型不漂浮在虚空
